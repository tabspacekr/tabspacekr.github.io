설치 방식

방식 A · 직접 배선

간편

각 방에서 보일러 밸브함(보통 주방 싱크대 하부)까지 220V 전선을 직접 배선하는 방식입니다.
TuYa 제어기(TGR86)가 직접 해당 방의 밸브 구동기 전원을 ON/OFF합니다.

기술 상세
▸ 전원: AC 220V (L + N) 2선
▸ 각 제어기(TGR86) → 밸브함까지 개별 배선 (방 수 × 2선)
▸ 밸브 구동기: 220V 전자식 인서트밸브 (경동 등)
▸ 제어 방식: TuYa 릴레이 접점 → 밸브 구동기 직접 구동
▸ 별도 릴레이보드 불필요

방식 B · 릴레이보드 (HA 연동)

고급

각 방의 TuYa 제어기(TGR86)가 Wi-Fi를 통해 공유기에 연결되고, Home Assistant 서버가 TuYa Integration을 통해 각 방의 온도 상태를 통합 수집합니다.

Home Assistant가 설정된 자동화 규칙에 따라 밸브함의 릴레이보드에 제어 신호를 전송하여 각 방의 인서트밸브를 개별적으로 ON/OFF합니다.
220V 전원 배선은 1쌍(L+N)만 밸브함까지 배선하면 됩니다.

기술 상세
▸ 배선: AC 220V (L + N) 1쌍만 밸브함까지
▸ TuYa 제어기(TGR86) → Wi-Fi → 공유기(Wi-Fi Router) 접속
▸ 공유기 → Home Assistant 서버 (TuYa Integration으로 각 방 온도 상태 수집)
▸ Home Assistant → 자동화 규칙에 따라 릴레이보드 제어 명령 전송
▸ 릴레이보드: HA 신호 수신 → 각 밸브별 220V ON/OFF 분배
▸ 밸브 구동기: 220V 전자식 인서트밸브 (경동 등)
▸ 필요 장비: 공유기, Home Assistant 서버, 릴레이보드
▸ 배선 최소화, 대신 추가 장비(서버, 릴레이보드) 비용 발생
